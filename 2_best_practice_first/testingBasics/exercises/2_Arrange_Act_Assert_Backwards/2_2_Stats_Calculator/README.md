# 2_2_stats_calculator

## Responsibilities

Of a selected array of numbers, calculate the following:

- The average
- The minimum
- The maximum
- The elements of the sequence

### Example

From the sequence = [2, 4, 21, -8, 53, 40];

- minimum value = -8,
- maximum value = 53,
- number of elements in the sequence = 6,
- average value = 18.666666666667

From the sequence = [100, 2, -3, -16, 1, 60, 81, 2, -3];

- minimum value = -16,
- maximum value = 100,
- number of elements in the sequence = 9,
- average value = 24.89

From the sequence = [5,5,5]

- minimum value = 5,
- maximum value = 5,
- number of elements in the sequence = 3,
- average value = 5

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.9. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
