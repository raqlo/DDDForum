# FizzBuzz

A simple FizzBuzz implementation using Bun and TypeScript.

## Prerequisites

Install Bun (if you don't have it already):
Bun already comes with the test runner. So it won't be necessary to install anything else.

## Getting Started

1. Install dependencies:

```bash
bun install
```

2. Run tests:

```bash
bun test
```

Or run tests in watch mode:

```bash
bun run test:dev
```

3. Run the project:

```bash
bun run index.ts
```
